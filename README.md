# Numpy Helper

```
a = np.array([[1,2],[3,4]])
b = np.array([[11,12],[13,14]])
np.dot(a,b) # `a` and `b` must be with the same dimensions
Will Produce:
[[37  40]
 [85  92]]

Calc:
[[1*11+2*13, 1*12+2*14],[3*11+4*13, 3*12+4*14]]
```
